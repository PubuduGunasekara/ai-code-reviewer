# 🤖 AI Code Review Assistant

> AI-powered pull request reviewer using GPT-4o + GitHub OAuth + React

## Status
🟡 **In Progress** — Week 1 of 6 (May 2026)

## Stack
Node.js · Express · OpenAI GPT-4o · GitHub OAuth · React · PostgreSQL · Redis · Docker

## Architecture
- GitHub OAuth login → connect your repos
- Select any open PR → AI reviews the diff
- Inline comments with severity scores
- Redis rate limiting + response caching
